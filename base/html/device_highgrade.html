제어기가 Client로 동작 시 서버 IP 주소와 포트 값 설정한다.

<기본값>

  • - Server 1 : 192.168.172.155 (Port 35000)

Transmission Method가 1 또는 2로 설정된 경우, 설정된 IP 주소로 신호등 상태 값이 전송된다

Device ID 장비의 고유 번호를 설정한다.
(범위 0 ~ 65535)
Traffic Signal Request
Traffic Signal Transmission 신호등 상태값의 서버 전송 여부를 설정한다
(0 : 전송금지, 1 : 전송)
Transmission Method
Transmission Method

신호등 상태정보의 서버 전송 방법을 설정한다.

  1. 0 : 서버로 전송하지 않음
  2. 1 : Method1, 신호등 상태 변경 될 경우 자동 전송
  3. 2 : Method2, 100msec 마다 자동으로 신호등 상태 전송
Device Reset
원격에서 함체 관리장치를 리셋 하기 위해 사용한다. 명령이 전송 된 후 해당 장비는 10초 이내로 리셋 한다.
