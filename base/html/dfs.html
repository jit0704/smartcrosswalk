DFS 설정

프로토콜 Data 비고
표시 문자 설정 숫자 0~255 표시, 문자 '/'로 행 구분, 행 당 최대 7글자(기본 표시), 최대 2행
문자의 표시 방식 설정 0 : 표시 안 함(default), 1 : 표시(기본), 2 : 흐르게 표시, 3 : 한 글자 씩 표시
표시 문자의 폰트 설정 0 : 기본 폰트(default), 1 : 폰트타입1, 2 : 폰트타입 2
표시 문자의 R, G, B 값 설정 '/'로 구분(default : 255/0/0 Red) Ex) "255/255/255" 흰색으로 문자 표시
표시 문자 점멸 모드 설정 0 : 점멸하지 않음(default) 1 : 점멸(점멸 간격은 Character blink interval에서 설정)
표시 문자 점멸 간격 설정 0 : 0.5초 간격 점멸(default) ~ 9 : 5초 간격 점멸 (0.5초씩 점멸 시간 증가)
문자 표시 시간 설정 0 : 다음 command가 올때까지 계속 표시(default) 1 : 1초간 표시 ~ 255초간 표시
문자 표시 후 blank 표시 시간 설정 0 ~ 255 : 설정 초간 blank, (0:default)
이미지의 표시 방식 설정 0 : 표시 안 함(default), 1 : 표시(기본), 2 : 흐르게 표시
표시할 이미지 인덱스 선택 0 : 0번 Index 이미지 표시(default) ~ 15 : 15번 Index 이미지 표시
표시 이미지 점멸 모드 설정 0 : 점멸하지 않음(default) 1 : 점멸 (점멸 간격은 Image blink interval에서 설정)
표시 이미지 점멸 간격 설정 0 : 0.5초 간격 점멸(default) ~ 9 : 5초 간격 점멸 (0.5초씩 점멸 시간 증가)
이미지 표시 시간 설정 0 : 다음 command가 올때까지 계속 표시(default) 1 : 1초간 표시 ~ 255 : 255초간 표시
이미지 표시 후 blank 표시 시간 설정 0 : 0초간 blank(default) ~ 255 : 255초간 blank
LED의 밝기 설정 0 : 센서 or 시간에 따라 설정된 밝기(default) 0x01 : 밝기 1로 설정 ~ 100 : 밝기 100으로 설정
정보 전송 완료/적용 및 설정값 반복 설정 0 : 설정값 반복(default) 1 : 설정값 1회 display ~ 255 : 설정값 255회 display
